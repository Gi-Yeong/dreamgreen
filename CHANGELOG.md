# 변경 사항 (Changelog)

## 📋 버전 규칙 (Semantic Versioning)

- **Major (X.0.0)**: 주요 기능 추가, 구조적 변경, 호환성 깨지는 변경
- **Minor (1.X.0)**: 새로운 기능 추가, UI/UX 개선, 성능 최적화
- **Patch (1.1.X)**: 버그 수정, 오타 수정, 작은 UI 조정
