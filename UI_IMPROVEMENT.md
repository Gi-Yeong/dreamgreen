## 🎨 CSS 스타일

### **메인 페이지 사용자 박스:**
```css
.user-info-box {
    position: fixed;          /* 고정 위치 */
    top: 20px;
    right: 20px;
    background: white;        /* 흰색 배경 */
    border-radius: 12px;
    padding: 12px 20px;
    box-shadow: 0 4px 15px rgba(0,0,0,0.15);
    border: 2px solid #667eea; /* 파란 테두리 */
    z-index: 1000;            /* 최상위 */
}

.user-name {
    color: #2c3e50;          /* 검은색 글자 */
    font-weight: bold;
}

.logout-btn {
    background: linear-gradient(135deg, #e74c3c 0%, #c0392b 100%);
    color: white;
    /* 호버 시 위로 살짝 올라감 */
}
```

### **업무 페이지 인라인 표시:**
```css
.user-info-inline {
    display: inline-block;
    background: white;
    color: #2c3e50;
    padding: 6px 15px;
    border-radius: 20px;
}

.user-name-small {
    color: #667eea;          /* 파란색 글자 */
}
```

## 🔍 동작 확인

### **1. 로그인 전:**
```
index.html 접속 시도
    ↓
자동으로 login.html로 이동
```

### **2. 로그인 성공:**
```
login.html에서 로그인
    ↓
index.html로 이동
    ↓
우측 상단에 [👤 minhee님] [로그아웃] 표시
```

### **3. 페이지 이동:**
```
index.html → app.html 이동
    ↓
app.html 헤더에도 사용자 정보 표시
    ↓
로그인 상태 유지됨
```

### **4. 로그아웃:**
```
로그아웃 버튼 클릭
    ↓
확인 다이얼로그 표시
    ↓
확인 시 login.html로 이동
```
